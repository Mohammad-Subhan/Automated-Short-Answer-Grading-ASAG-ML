# Automated Short Answer Grading (ASAG) — ML

This repository contains a Machine Learning approach to **Automated Short Answer Grading** using the Mohler et al. (2011) dataset, which consists of ~2,273 student answers across 79 Computer Science questions.

## Description

The project implements a regression-based grading system that combines semantic, lexical, and rubric-based features:
- **Semantic Features:** SBERT (`all-MiniLM-L6-v2`) embeddings to compute cosine similarity between student and reference answers.
- **Rubric Features:** Evaluation of "rubric coverage" by checking how well the student's answer covers individual sentences of the reference answer.
- **Lexical Features:** Keyword overlap (Jaccard similarity) and answer length ratios.
- **Models:** Evaluates Ridge Regression and Random Forest models using **Leave-One-Question-Out Cross-Validation (LOQO-CV)** to ensure generalization to unseen questions.

## Data
The project uses the Mohler dataset stored in the `data/` directory.
- `dataset.csv`: The raw dataset.
- `dataset_preprocessed.csv`: Precomputed features (generated by the notebook).
